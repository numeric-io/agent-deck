**Integration tests (require tmux):**

1. `TestSessionStart_CreatesTmuxSession` (TEST-03): Create instance with `NewInstance("test-start", "/tmp")`, set `Command = "sleep 60"`. Call `Start()`. Assert no error. Assert `inst.Exists() == true`. Assert `inst.GetTmuxSession() != nil`. Assert `inst.GetTmuxSession().Name()` is non-empty. Verify tmux session exists via `exec.Command("tmux", "has-session", "-t", inst.GetTmuxSession().Name())` returning no error. `defer func() { _ = inst.Kill() }()`.

2. `TestSessionStart_SetsStartingStatus` (TEST-03): Create instance with command. Call `Start()`. Immediately check `inst.Status == StatusStarting` (before grace period expires). `defer Kill()`.

3. `TestSessionStop_KillsAndSetsError` (TEST-04): Create instance, `Start()`, verify `Exists()`. Call `Kill()`. Assert `inst.Status == StatusError`. Assert `inst.Exists() == false`. Verify tmux session is gone via `exec.Command("tmux", "has-session", "-t", name)` returning an error.

4. `TestSessionStop_DoubleKill` (TEST-04): Create instance, `Start()`, `Kill()`. Call `Kill()` again. Assert no panic, error is acceptable (tmux session already gone).

5. `TestSessionStop_UpdateStatusAfterKill` (TEST-04): Create instance, `Start()`, `Kill()`. Call `UpdateStatus()`. Assert `StatusError` since tmux session no longer exists.

**Unit test (no tmux):**

6. `TestSessionStart_NilTmuxSession`: Create a bare `&Instance{}` without tmux initialization. Call `Start()`. Assert error contains "tmux session not initialized".

Use `require` for fatal assertions, `assert` for checks. Always `defer func() { _ = inst.Kill() }()` after successful `Start()`.
