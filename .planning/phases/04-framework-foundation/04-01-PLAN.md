    **testmain_test.go (INFRA-04):**
    - `package integration`
    - `TestMain(m *testing.M)`: calls `testutil.UnsetGitRepoEnv()`, sets `AGENTDECK_PROFILE=_test`, runs `m.Run()`, calls `cleanupIntegrationSessions()`, then `os.Exit(code)`
    - `cleanupIntegrationSessions()`: lists tmux sessions via `exec.Command("tmux", "list-sessions", "-F", "#{session_name}")`, kills any with prefix `agentdeck_inttest_`. Use ONLY this prefix, not broader patterns.

    **harness.go (INFRA-01):**
    - `package integration`
    - `TmuxHarness` struct with fields: `t *testing.T`, `sessions []*session.Instance`, `prefix string`
    - `NewTmuxHarness(t *testing.T) *TmuxHarness`: calls `t.Helper()`, `skipIfNoTmuxServer(t)`, generates prefix `fmt.Sprintf("inttest_%s_", sanitize(t.Name()))` where `sanitize` replaces `/` with `_` and strips non-alphanumeric chars. Registers `t.Cleanup(h.cleanup)`.
    - `CreateSession(title, projectPath string) *session.Instance`: creates instance via `session.NewInstance(h.prefix+title, projectPath)`, appends to `h.sessions`, returns instance.
    - `CreateSessionWithTool(title, projectPath, tool string) *session.Instance`: same but uses `session.NewInstanceWithTool`.
    - `cleanup()`: iterates sessions in reverse, calls `inst.Kill()` for each that `inst.Exists()`. Ignores errors (cleanup best-effort). Logs nothing (test output stays clean).
    - `SessionCount() int`: returns len(h.sessions).

    **poll.go (INFRA-02):**
    - `package integration`
    - `WaitForCondition(t *testing.T, timeout, poll time.Duration, desc string, condition func() bool)`: uses `time.After(timeout)` + `time.NewTicker(poll)`. Checks condition before first tick. On timeout: `t.Fatalf("timed out after %v waiting for: %s", timeout, desc)`.
    - `WaitForPaneContent(t *testing.T, inst *session.Instance, contains string, timeout time.Duration)`: wraps WaitForCondition with 200ms poll interval. Gets tmux session via `inst.GetTmuxSession()`, calls `CapturePaneFresh()`, checks `strings.Contains(content, contains)`. Handles nil tmux session gracefully (returns false, keeps polling).
    - `WaitForStatus(t *testing.T, inst *session.Instance, status session.Status, timeout time.Duration)`: wraps WaitForCondition with 200ms poll interval. Reads `inst.GetStatusThreadSafe()`.

    **Write tests first (RED), then implement (GREEN).** Tests for harness require tmux so use `skipIfNoTmuxServer`. Tests for polling can use a simple counter-based condition for the basic case (TestWaitForCondition_Success), and pane content / status tests require tmux.
