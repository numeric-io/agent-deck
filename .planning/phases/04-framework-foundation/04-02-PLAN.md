```go
type Status string
const (
    StatusRunning  Status = "running"
    StatusWaiting  Status = "waiting"
    StatusIdle     Status = "idle"
    StatusError    Status = "error"
    StatusStarting Status = "starting"
)

func NewInstance(title, projectPath string) *Instance
func (i *Instance) Start() error
func (i *Instance) Kill() error
func (i *Instance) Exists() bool
func (i *Instance) Restart() error
func (i *Instance) GetTmuxSession() *tmux.Session
func (i *Instance) GetStatusThreadSafe() Status
func (i *Instance) CreateForkedInstance(newTitle, newGroupPath string) (*Instance, string, error)
```

Key timing notes from research:
- Start() sets Status to StatusStarting. Grace period is 1.5s (instance.go:2263).
- UpdateStatus() returns early during grace period if tmux session doesn't exist yet.
- CapturePane has singleflight cache with 500ms TTL; use CapturePaneFresh() for assertions.
- Kill() sets Status to StatusError.
- Restart() on dead shell session: kills then recreates via Start(). Sets StatusWaiting if command provided, StatusIdle if not.
